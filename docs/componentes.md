# Diagrama de Componentes

![Diagrama de Componentes](./imagenes/diagrama-componentes.png)

## Descripción
Este diagrama muestra los principales componentes del sistema y la forma en que se comunican entre sí. Incluye elementos de la interfaz de usuario, módulos de validación, rutas o servicios de atención, lógica de negocio y persistencia de datos, organizados para soportar procesos como la gestión de pacientes, medicamentos y recordatorios.

## Justificación
El diagrama es coherente con la arquitectura del proyecto porque representa la separación entre presentación, control, lógica de negocio y acceso a datos. Esto permite entender cómo fluye la información dentro del sistema y cómo se integran los módulos responsables de cada funcionalidad.
